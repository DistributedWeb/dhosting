FAQ

What is a dPack vault?

A dPack vault is a collection of files that can be efficiently distributed to other peers on the dWeb network.

How do I create a dPack vault?

You can use the dBrowser, the official dPack command line CLI, or the dWeb's dFTP desktop app.

Can I self host dHosting?

Yes. The software that powers dHosting, dHost, is open source and easy to deploy.

You may also find dPack HTTP is a good choice.
